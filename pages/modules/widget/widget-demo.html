📦 Widget Incorporado

Código usado:
<div class="tokencafe-widget" data-owner="0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb" data-code="tc-20250116-143522-abc123"> </div>

O widget abaixo foi carregado dinamicamente via tokencafe-widget.min.js. Ele busca a configuração JSON em /widget/gets/<owner>/<code>.json e renderiza a interface de compra completa.

📋 Instruções de Uso

  1. Gere seu widget através da interface admin em /pages/modules/widget/widget-teste.html
  2. Baixe o JSON gerado e coloque em /widget/gets/<seu-endereço>/<código>.json
  3. Incorpore em qualquer site usando o snippet fornecido
  4. Pronto! O widget carrega automaticamente quando a página é aberta
💡 Dica: Este sistema permite que você crie múltiplos widgets (diferentes Sales, redes, tokens) e gerencie todos a partir de um único endereço owner.
