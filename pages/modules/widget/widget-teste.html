Etapa 3: Deploy no Remix

Siga estas instruções para fazer o deploy do seu contrato no Remix IDE.

Importante: Certifique-se de ter BNB suficiente na sua carteira para pagar o gas do deploy.

  • Certifique-se de ter BNB na sua carteira (para gas)
  • Tenha o arquivo .sol baixado ou código copiado
  • Carteira MetaMask configurada para BSC Testnet
Dica: Você pode obter BNB testnet gratuito no BSC Faucet

  1. Acesse remix.ethereum.org
  2. Aguarde o carregamento completo da interface
  3. Familiarize-se com as abas: File Explorer, Solidity Compiler, Deploy & Run

  1. No File Explorer, clique no ícone "+" para criar novo arquivo
  2. Nomeie o arquivo como TokenSale.sol
  3. Cole todo o código do contrato gerado
  4. Salve o arquivo (Ctrl+S)
Atenção: Certifique-se de colar o código completo, incluindo a licença e imports.

  1. Clique na aba "Solidity Compiler" (ícone do Solidity)
  2. Verifique se a versão do compilador é 0.8.x
  3. Clique em "Compile TokenSale.sol"
  4. Aguarde a compilação (deve aparecer um check verde)
Sucesso: Se aparecer um check verde, a compilação foi bem-sucedida!

  1. Clique na aba "Deploy & Run Transactions"
  2. Em "Environment", selecione "Injected Provider - MetaMask"
  3. Conecte sua carteira MetaMask quando solicitado
  4. Verifique se está conectado à BSC Testnet (Chain ID: 97)
  5. Confirme que seu endereço aparece em "Account"
Importante: Certifique-se de estar na BSC Testnet, não na Mainnet!

  1. Em "Contract", selecione "TokenSale"
  2. Clique no botão laranja "Deploy"
  3. Confirme a transação na MetaMask
  4. Aguarde a confirmação na blockchain (pode levar alguns minutos)
  5. O contrato aparecerá na seção "Deployed Contracts"
Gas Fee: O deploy consumirá BNB como taxa de gas. Certifique-se de ter saldo suficiente.

  1. Após o deploy bem-sucedido, vá para "Deployed Contracts"
  2. Clique no ícone de "copy" ao lado do nome do contrato
  3. O endereço será copiado para sua área de transferência
  4. Guarde este endereço - você precisará dele na próxima etapa
Pronto! Seu contrato foi deployado com sucesso. Agora você pode prosseguir para a próxima etapa.
Dicas Importantes:
  • Mantenha o endereço do contrato em local seguro
  • Você pode verificar o contrato no BSCScan Testnet
  • Lembre-se de transferir tokens para o contrato antes de iniciar as vendas
