🚨 Node.js NÃO Instalado

Para usar o TokenCafe completo, você precisa instalar o Node.js primeiro

🔍 Problema Identificado

Node.js não está instalado no seu computador. Por isso o comando npm start não funciona.

⚡ Solução Rápida (5 minutos)

  1. Baixar Node.js: Clique no botão verde abaixo
  2. Instalar: Execute o arquivo baixado (aceite tudo)
  3. Reiniciar: Feche e abra o PowerShell novamente
  4. Testar: Execute os comandos do passo 4
📥 BAIXAR NODE.JS (Windows 64-bit)

📋 Passo a Passo Detalhado

1

Baixar Node.js

Acesse nodejs.org e baixe a versão LTS (recomendada)

Ir para nodejs.org
2

Instalar Node.js

Execute o arquivo .msi baixado e aceite todas as configurações padrão

Vai instalar o Node.js e npm automaticamente
3

Verificar Instalação

Abra um novo PowerShell e execute:

node --version
npm --version
4

Iniciar TokenCafe

Navegue até o projeto e execute:

cd "C:\Users\User\Desktop\cafe\tokencafe"
npm install
npm start

🎯 Resultado Esperado

Após seguir os passos, você verá algo como:

🚀 TokenCafe Server iniciado com sucesso!
🌐 Servidor: http://localhost:3001
📊 Dashboard: http://localhost:3001/dashboard

Então acesse http://localhost:3001 no navegador!

🛠️ Problemas Comuns

Causa: Node.js não foi instalado corretamente ou PATH não foi configurado.
Solução:
  1. Reinstale o Node.js
  2. Reinicie o computador
  3. Abra um novo PowerShell

Causa: PowerShell sem permissões administrativas.
Solução:
  1. Clique direito no PowerShell
  2. Selecione "Executar como administrador"
  3. Execute os comandos novamente

Causa: Cache do npm corrompido ou conexão de internet.
Solução:
npm cache clean --force
npm install
