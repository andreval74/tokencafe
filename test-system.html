2. Iniciar Servidor
npm run dev # ou npm start
3. Acessar Aplicação

🏠 Frontend: http://localhost:3001

📊 Dashboard: http://localhost:3001/dashboard

4. APIs Disponíveis

🔐 Auth: /api/auth/*

🧩 Widgets: /api/widgets/*

📈 Analytics: /api/analytics/*
