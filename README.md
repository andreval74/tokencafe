# TokenCafe — Democratizando Web3 🇧🇷

**Plataforma modular para criação e deploy de tokens EVM (BSC Testnet / Sepolia)**

## 🚀 Quick Start

```bash
# 1. Instalar dependências
npm install

# 2. Rodar servidor PHP (portal)
npm run serve
# Acesse: http://localhost:8000

# 3. Rodar API Node.js (opcional)
npm run dev
# Acesse: http://localhost:3000
```

## 🏗️ Stack

- **Frontend**: PHP 8+ + JavaScript ES Modules + Bootstrap 5
- **Backend**: Node.js + Express + MongoDB
- **Contratos**: Solidity ^0.8.0 + Hardhat + OpenZeppelin
- **Redes**: BSC Testnet / Sepolia
